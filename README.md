# Log Forensics Project

## Description
Analyze log files to detect suspicious activity:
- High number of requests from a single IP
- Error responses (4xx, 5xx)

## Files
- analyzer.py → Python script for analysis
- logs.txt → Sample log data
- Notebook → Colab notebook with analysis

## How to Run
1. Upload logs.txt
2. Run analyzer.py
3. View output for suspicious IPs and errors
